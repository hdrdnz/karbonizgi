# 🌱 Karbonİzgi

**Karbonİzgi**, bireysel ve kurumsal kullanıcıların karbon ayak izini hesaplayıp azaltmalarına yardımcı olan, sürdürülebilir yaşamı destekleyen kapsamlı bir mobil ve web uygulamasıdır.

## 🚀 Özellikler

- 📊 **Karbon Ayak İzi Hesaplama**  
  - Bireyler için ulaşım, enerji tüketimi ve beslenme alışkanlıklarına göre skor hesaplama  
  - Şirketler için enerji kaynakları, üretim verimliliği, atık yönetimi gibi parametrelere dayalı hesaplama

- 🤖 **Yapay Zekâ Destekli Tavsiye Sistemi**  
  - OpenAI tabanlı özel fine-tuning modeli ile test sonrası kişiselleştirilmiş öneriler  
  - “Karbon ayak izim yüksek mi?” gibi sorulara dinamik yanıtlar

- 🌍 **Farkındalık ve Aksiyon Planı**  
  - Kullanıcıya sadece bilgi değil, aynı zamanda uygulanabilir aksiyon önerileri sunar  
  - Gelişmiş raporlama ve sürdürülebilirlik ölçütleri ile bilinçli dönüşüm sağlar

---

## 🛠️ Kullanılan Teknolojiler

- Backend: **Go (Golang)** 
- Veritabanları: **MySQL**  
- Diğer: **OpenAI GPT Fine-Tuning**, **Docker**

---

## 🧠 Nasıl Çalışır?

1. Kullanıcı test sorularını yanıtlar
2. Sistem ISO 14064 standardına göre karbon ayak izini hesaplar
3. AI destekli sohbet ekranında öneriler sunulur
4. Grafik kısmında kişiselleştirilmiş grafik detaylarına ulaşır.
5. Profil kısmında genel skor durumunu ve sıralamasını görür.
